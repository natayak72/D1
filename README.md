**Порядок установки**

0. **Подготовить виртуальное окружение**<br>
    python -m venv venv
1. **Установить зависимости <br>**
   pip install -r requirements.txt
2. **Запустить сервер** <br>
    python manage.py runserver 

**Использование**<br>
Стартовая страница: 
http://127.0.0.1:8000/first/
